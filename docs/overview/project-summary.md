# プロジェクト概要

Trulay は、「信頼 × 希望」を軸に、個人で仕事を受ける人と企業が安心してつながれるサービスを目指すプロジェクトです。

初期段階では対象を広げすぎず、フェーズ1 ではフリーランス向けの営業管理体験に集中します。

## フェーズ1 の方向性

フェーズ1 では、企業側アカウントを作らず、フリーランスが個人で以下を管理できる状態を目指します。

- 案件情報
- 企業情報
- 担当者情報
- 応募状況・商談状況
- メモ
- 次回アクション

## 将来の方向性

- フェーズ2: 企業アカウント・案件共有
- フェーズ3: マッチング強化
- フェーズ4: 信頼性強化
