# Phase Plan

Trulay の開発フェーズを整理します。

詳細なタスク一覧は [Project Roadmap](./project-roadmap.md) にまとめます。

## Phase 1: フリーランス向け営業管理

フリーランスが個人で案件、企業情報、応募状況、商談状況、メモ、次回アクションを管理できる状態を作ります。

## Phase 2: 企業アカウント・案件共有

企業がアカウントを作成し、案件を共有できる状態を作ります。

## Phase 3: マッチング強化

フリーランスと企業が、より自分に合う相手・案件と出会える体験を強化します。

## Phase 4: 信頼性強化

本人確認、企業認証、レビューなど、安心してつながるための仕組みを強化します。
